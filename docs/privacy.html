Privacy Policy

Last Updated: February 24, 2026

⚠️ Demo Project Notice: This is a demonstration prototype built for a case study exercise. It uses mock data and is not affiliated with Angi Inc. No real personal information is collected or stored.

1. Information Collection

This demo application may collect the following information when you interact with it:

  • Questions you ask about home services
  • Service categories you search for
  • Zip codes you provide for location-based searches
  • Information you provide when requesting mock quotes (name, zip code, service description)

2. How We Use Information

Information is used solely to:

  • Demonstrate the functionality of an MCP (Model Context Protocol) server
  • Return relevant mock data from our demonstration database
  • Show how AI platforms could integrate with home service APIs

3. Data Storage

No data is permanently stored. This is a stateless demonstration API:

  • All interactions are processed in real-time
  • No database stores user queries or personal information
  • All "professionals" and "quotes" are mock data for demonstration purposes
  • Server logs may temporarily capture requests for debugging but are not retained long-term

4. Mock Data

All data returned by this API is fictional and for demonstration purposes only:

  • Professional names, businesses, and contact information are not real
  • No actual services will be performed
  • No real quotes will be generated
  • Lead IDs are randomly generated and not tracked

5. Third-Party Services

This demo is hosted on:

  • Railway (server hosting) - Subject to Railway's privacy policy
  • GitHub Pages (demo interface) - Subject to GitHub's privacy policy

When accessed via ChatGPT Custom GPT or Claude Desktop, interactions are also subject to OpenAI's or Anthropic's privacy policies.

6. Cookies and Tracking

This demo does not use cookies or tracking technologies. Standard server logs may record:

  • IP addresses
  • Request timestamps
  • API endpoints accessed

7. Security

While this is a demo with no authentication:

  • All connections use HTTPS encryption
  • No sensitive personal information should be provided
  • This is not a production system and should not be used for real service requests

8. Children's Privacy

This demo is not intended for children under 13 and does not knowingly collect information from children.

9. Your Rights

Since no data is stored, there is no personal data to access, modify, or delete. Each interaction is independent and ephemeral.

10. Changes to This Policy

This privacy policy may be updated to reflect changes in the demo. The "Last Updated" date at the top indicates when changes were made.

12. Disclaimer

This is a prototype demonstration and is not affiliated with, endorsed by, or representing Angi Inc. or HomeAdvisor. All data is fictional. No real services, professionals, or quotes are provided. This project was created solely for educational and demonstration purposes as part of a case study exercise.
